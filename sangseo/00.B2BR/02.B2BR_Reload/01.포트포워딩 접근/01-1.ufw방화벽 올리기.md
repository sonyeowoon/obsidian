# ufw 설치하기
ufw에 대한 사용법은 [이곳](https://blog.codingcat.kr/80)에 잘 정리되어 있다.
개념은 [이곳](https://blog.itcode.dev/posts/2021/09/04/firewall-with-ufw)에서 확인하도록 하자.
기본적으로 ufw는 설치되어있지 않다.
```
apt-get install ufw
```
위 명령어를 통해 설치하도록 하자.
중간에 'Y'를 눌러서 계속 진행하도록 한다.
