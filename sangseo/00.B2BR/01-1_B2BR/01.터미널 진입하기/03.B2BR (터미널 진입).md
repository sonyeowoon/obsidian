# sudo 권한 얻기 & 업데이트
![[Pasted image 20240407205124.png]]
```
apt-get update
apt-get install sudo
```
지금 상황에서는 sudo가 작동하지 않을 것이다.
이러한 경우에는 위 명령어들을 사용하면 된다.
